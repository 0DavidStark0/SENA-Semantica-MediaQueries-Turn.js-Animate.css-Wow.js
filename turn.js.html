Turn.js

Descripcion de Turn.js y ejemplos

Descripcion

Turn.js es una biblioteca de JavaScript que hará que su contenido se vea como un libro o revista real utilizando todas las ventajas de HTML5. La web se está volviendo hermosa con nuevas interfaces de usuario basadas en HTML5; turn.js es la mejor opción para una revista, libro o catálogo basado en HTML5.

Caracteristicas

  • Funciona en iPad y iPhone.
  • API simple, hermosa y poderosa.
  • Permite cargar páginas dinámicamente a través de solicitudes Ajax.
  • Contenido puro HTML5/CSS3.
  • Dos efectos de transición.
  • Funciona en navegadores antiguos como IE 8 con turn.html4.js

Requisitos

jQuery 1.3 o superior.

Compatibilidad con navegadores

  • Safari 5
  • cromo 16
  • firefox 10
  • IE 10, 9, 8

Dispositivos

  • Todo iOS (iPad, iPhone, iPod)
  • Android (Cromo para Android)

Mejoras

Turn.js 4 incluye un conjunto de mejoras de rendimiento significativas en su núcleo.

  • Los efectos ahora son bastante más fluidos en la plataforma del navegador.
  • La nueva composición DOM garantiza el mismo rendimiento sin importar la cantidad de páginas.

Complementos

turn.html4.js: la versión HTML4 de turn.js.
zoom.js: la nueva función de zoom de turn.js.
tijeras.js - Corta una página en dos partes para turn.js.
hash.js: controla el historial de navegación mediante pushState y URI hash.
